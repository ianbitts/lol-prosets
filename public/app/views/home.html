Welcome to Prosets!

This site is designed to help summoners design item sets, look at anyones' recent ranked match history, and modify those items sets from match history.

How to use the Match History feature

This page will allows the user to search players by their summoner name and view their match history. Various stats of the match will be displayed. There will be a Get Item Set button that will direct users to the Item set Builder page and preload the selected item set.


1. Enter Summoner Name
2. Select the region they are in
3. If the name is valid, a list of their most recent matches should populate the screen to view
4. If the champions are clicked from the Match History by Champion window, the screen will repopulate the screen with recent matches by that hero if there are any.
