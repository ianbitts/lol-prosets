Welcome to Pro Sets!

This website makes it easy to look up your favorite players and build item sets based on their recent match history. After selecting a match, the app uses timeline data to break the player's purchases into starting, early game, mid game and late game picks. Then, you can download the sets for use in your own games!

Get Started

How to use the Match History feature

The Match History page allows the user to search for players by their summoner name and view their recent match history for ranked games. To get started, follow the instructions below:

Enter a Summoner Name and select an appropriate region.


If the name is valid, a list of the player's most recent matches will populate the screen. The Match History by Champion window will allow users to filter recent matches by specific champions.

Upon a successful search, the screen should populate with recent matches that look like this:

The Get Item Set button will redirect to the Item Set Builder page where the player's items will be broken into item set blocks based on when the items were purchased.


How to use Item Set Builder

This feature will let summoners build item sets to be used in game! You can build your item set from scratch or import it from match history.

First, select the map that will be used for the item set:


Next, select the champion you want to build the item set for in Champions window. If you want to build a general item set, do not select a champion. You may add additional items to your item set by dragging the desired item from the Item Panel on the right side of the screen into one of your item blocks. Items in the item panel may be filtered by category, price, or name by using the checkboxes and search bar within.


The New Item Set window is where you will build your item set. You can add more groups of items to one item set by clicking Add Group button. Only one block is required to build an item set.

Once finished, click the Download Item Set button. A .zip file will be downloaded to your default downloads directory. Unzip the file, and then copy Config directory contained within into your League of Legends directory. The default League of Legends directory is located at
