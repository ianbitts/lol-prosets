Welcome to Pro Sets!

This website makes it easy to look up your favorite players and build item sets based on their recent match history. After selecting a match, the app uses timeline data to break the player's purchases into starting, early game, mid game and late game picks. Then, you can download the sets for use in your own games!

How to use Item Set Builder

This feature will let summoners build item sets to be used in game! You can build your item set from scratch or import it from match history.

1. First, select the map for to build the item set for. It should look like this:



2. Select the champion you want to build the item set for in Champions window. If you want do build a general item set, don't select any.

3. The Items window is where you can view the items to choose from to apply to your set. the search box and the various checkbox items help you to filter the list to find them faster.



4.To add an item to your set, click and drag the item you want and drop it into the item block.



5. The New Item Set window is where you will build your item set. You can add more groups of items to one item set by clicking Add Group button. Only one block is required to build an item set.
